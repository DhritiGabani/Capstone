# Dorsiflexx
```markdown
Ankle Rehabilitation Monitoring System

A React Native mobile application for tracking ankle rehabilitation exercises using dual-IMU sensors and on-device ML inference.

## Prerequisites

- Node.js (v16+)
- npm or yarn
- Expo Go app (for testing on physical device)

## Installation

```bash
# Clone the repository
git clone https://github.com/your-username/dorsiflexx.git
cd dorsiflexx

# Install dependencies
npm install
```

## Running the App

```bash
# Start the development server
npx expo start

# Options:
# - Press 'i' for iOS simulator (requires Mac + Xcode)
# - Press 'a' for Android emulator (requires Android Studio)
# - Scan QR code with Expo Go app on your phone
```

The database will initialize automatically on first launch.

## Tech Stack

- **Frontend**: React Native with Expo
- **Database**: SQLite (Expo SQLite)
- **ML Model**: TensorFlow Lite (on-device inference)
```
